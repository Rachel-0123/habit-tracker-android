# Habit Tracker

A modern Android habit tracking app built with Kotlin and Jetpack Compose.

Track daily habits, monitor streaks, and visualize monthly progress through a clean and colorful UI.

---

## Features

### Daily Habit Tracking
- Create and manage habits
- Mark habits as completed each day
- Animated completion states
- Habit streak tracking

### Monthly Calendar View
- Scrollable month calendar
- Visual overview of completed habits
- Color-coded habit indicators
- Tap dates to view completed habits

### Habit Customization
- Custom habit colors
- Edit existing habits
- Delete habits
- Persistent local storage

### Modern UI
- Built entirely with Jetpack Compose
- Material 3 design
- Smooth animations
- Responsive layouts
- Bottom navigation support

---

## Tech Stack

- Kotlin
- Jetpack Compose
- Material 3
- Android ViewModel
- DataStore Preferences
- Kotlin Serialization

---

## Architecture

The app follows a simple MVVM architecture:

- **UI Layer**
    - Compose screens and components

- **ViewModels**
    - UI state management
    - Business logic

- **Data Layer**
    - DataStore persistence
    - Serialized habit storage
